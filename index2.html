  • 在 settings.py 的 install_apps 中注册
  INSTALLED_APPS = [
      'corsheaders'
  ]
  • 在settings.py 的 middleware 中注册
  MIDDLEWARE = [
      'corsheaders.middleware.CorsMiddleware', # 一定要在commonmiddleware之上
    'django.middleware.common.CommonMiddleware',
  ]
  • 在 settings.py 的任意位置 添加
  CORS_ORIGIN_ALLOW_ALL = True
  • 完成django 跨域

Python循环-while

Permalink: 2019-08-08 18:44:00+02:00 by 商伟 in Python tags: Python

循环语句介绍

<1>软件开发中循环的使用场景

跟媳妇承认错误，说一万遍"媳妇儿，我错了"

    print("媳妇儿，我错了")
    print("媳妇儿，我错了")
    print("媳妇儿，我错了")
    ...(还有99997遍)...

使用循环语句一句话搞定

    i = 0
    while i < 10000:
        print("媳妇儿，我错了")
        i += 1

<2>总结

  • while和if的用法基本类似，区别在于：if 条件成立，则执行一次； while 条件成立，则重复执行，直到条件不成立为止。
  • 一般情况下，需要多次重复执行的代码，都可以用循环的方式来完成
  • 循环不是必须要使用的，但是为了提高代码的重复使用率，所以有经验的开发者都会采用循环
