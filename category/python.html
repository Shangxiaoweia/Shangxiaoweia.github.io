Python循环-while

Permalink: 1000-01-05 18:44:00+00:09 by 商伟 in Python tags: Python

循环语句介绍

<1>软件开发中循环的使用场景

跟媳妇承认错误，说一万遍"媳妇儿，我错了"

    print("媳妇儿，我错了")
    print("媳妇儿，我错了")
    print("媳妇儿，我错了")
    ...(还有99997遍)...

使用循环语句一句话搞定

    i = 0
    while i < 10000:
        print("媳妇儿，我错了")
        i += 1

<2>总结

  • while和if的用法基本类似，区别在于：if 条件成立，则执行一次； while 条件成立，则重复执行，直到条件不成立为止。
  • 一般情况下，需要多次重复执行的代码，都可以用循环的方式来完成
  • 循环不是必须要使用的，但是为了提高代码的重复使用率，所以有经验的开发者都会采用循环
