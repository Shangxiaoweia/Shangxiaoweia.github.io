创建仓库

网页的右上角

定义全局变量

git config --global user.name "Shangxiaoweia"
git config --global user.email "shangxiaoweia@126.com"

从码云上拿代码

git clone 网址

cd myvue

git pull

更新自己的

git add -A

git commit -m '自定义介绍'

git push

建仓库

git init

git add -A

git commit - m '自定义介绍'

git config --global user.name "自定义名字"

git commit -m '自定义介绍'

git remote add origin ‘网址’

git push -u origin master
