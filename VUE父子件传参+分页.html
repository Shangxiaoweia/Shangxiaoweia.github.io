VUE父子件传参+分页

Permalink: 1000-01-01 18:44:00+00:09 by 商伟 in VUE tags: VUE

直接上代码、然后解释：

父组件代码

<template>
  <div id="fu">
    这是父组件
    <zi :name1=data @da="add"></zi>
  </div>
</template>
<script>
  import zi from '../components/zi'

  export default {
    name: "fu",
    data() {
      return {
        data: [1, 2, 3, 4]
      }
    },
    components: {
      'zi': zi
    },
    methods: {
      add(p) {
        alert(p)
      }
    }
  }
</script>
<style scoped>
</style>

子组件代码

<template>
  <div id="zi">
    这是子组件: <br>
    <br>
    <button v-for="p in name1" @click="aa(p)">{{p}}</button>
  </div>
</template>

<script>
  export default {
    name: "zi",
    data() {
      return {
        data: [],
      }
    },
    methods: {
      aa(p) {
        this.$emit('da', p)
      }
    },
    props: ['name1'],

  }
</script>

<style scoped>

</style>

组件嵌套分为三部分

1）调用

  import zi from '../components/zi'

2）注册

    components: {
      'zi': zi
    },

3）引用

    <zi></zi>

父组件给子组件传参

绑定的值可以直接传到子组件

    <zi :name1=data></zi>

子组件接收父组件的值

用props就可以直接接收父组件传过来的值，获取的值可以再子组件里直接使用

    props: ['name1'],

子组件给父组件传参

利用点击事件进行传参

    methods: {
      aa(p) {
        this.$emit('da', p)
      }
    },

父组件接收子组件的值

    <zi :name1=data @da="add"></zi>

    methods: {
      aa(p) {
        this.$emit('da', p)
      }
    },
