Django上传图片

Permalink: 2019-08-08 18:44:00+02:00 by 商伟 in Django tags: Django

最简单的保存图片与展示

  • 在Django里面

  • 根目录创建 static 文件夹

  • settings

STATIC_URL = '/static/'

STATICFILES_DIRS = (
    os.path.join(BASE_DIR, 'static'),
)
  • views
from datetime import datetime
class Addimg(APIView):
    def post(self, request):
       data = request.data
        img = data.get('img')
        now = datetime.strftime(datetime.now(), '%Y%m%d%H%M%S%f')
        with open('static/imgs/' + now + img.name, 'wb') as f:
        f.write(img.read())
        return Response({'code': 200, 'message': '成功'})

  • 看图片 或者在网页显示

  • 直接访问链接就可以 

http://127.0.0.1:8000/static/图片名
