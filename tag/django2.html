Permalink: 2019-08-08 18:44:00+02:00 by 商伟 in Django tags: Django

1)安装pelcan和markdown

pip install pelican  markdown --upgrade

2)创建一个文件夹用来作为博客的目录

mkdir 博客目录

3)cd到该目录下运行命令生成博客的框架

cd 博客目录
pelican-quickstart
