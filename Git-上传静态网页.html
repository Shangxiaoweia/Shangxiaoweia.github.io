Git-上传静态网页

Permalink: 1000-01-01 18:44:00+00:09 by 商伟 in GIT tags: GIT
首先申请 Github 免费静态内容空间，在github.com注册一个账号，然后创建一个新的仓库

仓库名称一定要定义成 账号名.github.io，这种形式，注意账号名一定要和你的github当前登录的账号吻合

这样你的静态内容空间就已经创建好了，在浏览器输入你的 https://你的账号.github.io/ 就可以访问了。
将之前打包好的public中的文件push到刚刚创建好的github仓库中去
访问静态空间网址[https://shangxiaoweia.github.io](https://shangxiaoweia.github.io/)
就这个样子结束了,搞定收工
