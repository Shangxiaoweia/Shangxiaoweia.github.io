js-if语句

Permalink: 1000-01-01 18:44:00+00:09 by 商伟 in JavaScript tags: JavaScript
定义
let a = 1,b = 2;
console.log(a)
console.log(b)
方法一
if(a > b){
  console.log('a大')
}else{
  console.log('b大')
}
方法二
if(a > b) console.log('a大')
if(a < b) console.log('b大'
方法三
if(a > b) console.log('a大')
else console.log('b大')
方法四
console.log(a>b ? 'a大' : 'b大')
